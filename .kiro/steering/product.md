# Product Overview

Lisa's Italian Restaurant is a single-screen Android app showcasing an authentic Italian restaurant located in Dublin, CA. This is a student project from the Android Basics Nanodegree Program (Project 1: Build a Single Screen App).

## Core Purpose
- Display restaurant information in an attractive, user-friendly format
- Provide interactive contact methods (phone, email, website, address)
- Showcase Italian cuisine through multiple food images
- Demonstrate fundamental Android development skills

## Business Details
- **Restaurant**: Lisa's Italian Restaurant
- **Location**: 104 Grafton St, Dublin, CA 94568
- **Specialty**: Authentic Italian food with party reservations
- **Hours**: Daily 11AM-10PM

## Key Features
- Responsive ScrollView layout for restaurant information
- Clickable contact elements (phone, email, website, address)
- Multiple Italian food images for visual appeal
- Material Design theming with Italian-inspired colors
- Full accessibility support with content descriptions
